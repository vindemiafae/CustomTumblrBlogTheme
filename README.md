# customtumblrblogtheme

Enchanted woodland Tumblr theme experiment.

## Files

- `theme.html` — paste this into Tumblr's Edit HTML panel.
- `rowanwood-overrides.css` — the custom woodland styling separated out for easier editing.

## First setup

1. Commit the original pasted theme as your backup.
2. Add `theme.html`.
3. In Tumblr Customize, upload a forest/lily pond image to **Forest Background**.
4. Change colors in `:root` first before editing individual selectors.

## Design direction

Forest + water lilies + fireflies. Keep it magical, readable, and not too visually crowded.
